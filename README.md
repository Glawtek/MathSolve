# MathSolve
MathSolve adalah package NPM beguna untuk menyelesaikan soal Matematika
## Install
```npm install mathsolve```
## Contoh
```js
const msolve = require("mathsolve")
console.log(msolve.luas.persegi(20))
```
Output
```
L = s x s
L = 20 x 20
L = 400
```
